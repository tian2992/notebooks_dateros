Introducción¶

Este es el primero de varios talleres introductorios al procesamiento y limpieza de datos. En este estaremos usando como ambiente de trabajo a Jupyter, que permite crear documentos con cdigo y prosa, adems de almacenar resultados de las operaciones ejecutadas (clculos, graficas, etc). Jupyter permite interactuar con varios lenguajes de programacin, en este, usaremos Python, un lenguaje de programacin bastante simple y poderoso, con acceso a una gran variedad de librerias para procesamiento de datos. Entre estas, est Pandas, una libreria que nos da acceso a estructuras de datos muy poderosas para manipular datos.

Comenzemos entonces!

Con esto podemos ver ya unas propiedades muy interesantes. Las series estn basadas en el concepto estadistico, pero incluyen un ttulo (del eje), un ndice (el cual identifica a los elementos) y el dato en s, que puede ser numerico (float), string unicode (texto) u otro tipo de dato.

Las series estan basadas tambien en conceptos de vectores, asi que se pueden realizar operaciones vectoriales en las cuales implicitamente se alinean los indces, esto es muy util por ejemplo para restar dos columnas, sin importar el tamao de ambas, automaticamente Pandas unir inteligentemente ambas series. Puedes tambien obtener elementos de las series por su valor de ndice, o por un rango, usando la notacin usual en Python. Como nota final, las Series comparten mucho del comportamiento de los NumPy Arrays, haciendolos instantaneamente compatibles con muchas librerias y recursos. https://pandas.pydata.org/pandas-docs/stable/dsintro.html#series

Pasemos ahora a DataFrames, como nuestro muni_data DataFrame. Los DataFrames son estructuras bi-dimensionales de datos. Son muy usadas porque proveen una abstraccin similar a una hoja de calculo o a una tabla de SQL. Los DataFrame tienen ndices (etiquetas de fila) y columnas, ambos ejes deben encajar, y el resto ser llenado de datos no validos.

Por ejemplo podemos unir ambas series y crear un DataFrame nuevo, usando un diccionario de Python, por ejemplo. Tambien podemos graficar los resultados.

## Esto es una funcion en Python, con def definimos el nombre de esta funcion, 'clean_q'
## esta recibe un objeto de entrada.

def clean_q(input_object):
    from re import sub  ## importamos la funcin sub, que substituye utilizando patrones
    ## https://es.wikipedia.org/wiki/Expresin_regular
    
    ## NaN es un objeto especial que representa un valor numrico invalido, Not A Number.
    if input_object == NaN:
        return 0
    inp = unicode(input_object) # De objeto a un texto
    cleansed_q = sub(r'Q\.','', inp) # Remueve Q., el slash evita que . sea interpretado como un caracter especial
    cleansed_00 = sub(r'\.00', '', cleansed_q) # Igual aqui
    cleansed_comma = sub(',', '', cleansed_00)
    cleansed_dash = sub('-', '', cleansed_comma)
    cleansed_nonchar = sub(r'[^0-9]+', '', cleansed_dash)
    if cleansed_nonchar == '':
        return 0
    return cleansed_nonchar
